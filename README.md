# ce-backend-lambda-exp

AWS clojure lambda-exp

## Installation

Fill in the required field in `create.sh`
```
./create.sh
```

## Usage

```
./invoke.sh
```

### AWS CodeBuild
use `ubuntu/java/openjdk-8` and `buildspec.yml` to build

### AWS CloudFormation
use `template.yml` to deploy
